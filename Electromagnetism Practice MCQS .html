Basic Concepts of Magnetism and Magnetic Fields

Fundamental Concepts:

Magnetic field originates from: North pole and terminates at South pole. Magnetic field lines are continuous closed loops.

Key Learning Trick:

Remember: Field lines come OUT of North and go INTO South. Outside magnet: N→S, Inside magnet: S→N.

Current Carrying Conductors:

When current flows through conductor, it produces magnetic field. Right hand rule: Grasp wire with thumb in current direction, fingers curl in field direction.

$\text{For straight wire: } B = \frac{\mu_0 I}{2\pi r}$

Table: Magnetic Field Due to Moving Charges

Charge State Magnetic Field Explanation
At Rest No magnetic field Only electric field exists
Uniform Motion Produces magnetic field $B = \frac{\mu_0 q v}{4\pi r^2}$
Accelerated Motion Produces changing magnetic field Also emits electromagnetic waves

Parallel and Anti-parallel Currents:

  • Same direction currents: Attract each other
  • Opposite direction currents: Repel each other
  • Force per unit length: $F/L = \frac{\mu_0 I_1 I_2}{2\pi d}$
MCQ Solving Strategy:

For current direction questions: Use right hand rule. For force direction between wires: Like currents attract, unlike currents repel.

Ampere's Circuital Law and Permeability

Ampere's Law Simplified:

For symmetrical cases: $B \cdot 2\pi r = \mu_0 I$ where B is magnetic field, r is distance, I is current.

$\text{Magnetic field around wire: } B = \frac{\mu_0 I}{2\pi r}$

Permeability Concepts:

  • $\mu_0$ = Permeability of free space = $4\pi \times 10^{-7}$ Tm/A
  • $\mu_r$ = Relative permeability = $\mu/\mu_0$
  • For vacuum/air: $\mu_r = 1$
  • For iron: $\mu_r \approx 5000$
Equation Interrelations:

$B \propto I$ (direct), $B \propto 1/r$ (inverse), $B \propto \mu$ (permeability). If current doubles, field doubles. If distance triples, field becomes 1/3.

Special Cases:

  1. Inside wire ($r < R$): $B \propto r$ (increases linearly from center)
  2. Outside wire ($r > R$): $B \propto 1/r$ (decreases inversely)
  3. At surface ($r = R$): Maximum field
MCQ Pattern Recognition:

Most MCQs ask: "If I changes to 2I and r changes to r/2, find new B." Solution: $B_{new} = B_{old} \times \frac{I_{new}}{I_{old}} \times \frac{r_{old}}{r_{new}}$

Ampere's Law MCQs (25)

Solenoid: Magnetic Field and Applications

Solenoid Field Equation:

For ideal solenoid (length >> diameter): $B = \mu_0 n I$ where $n = N/L$ = turns per unit length.

$\text{Field in solenoid: } B = \frac{\mu_0 N I}{L}$

Right Hand Rules:

  1. Right hand grip rule: Curl fingers in current direction, thumb shows North pole
  2. End rule: Looking at end, if current clockwise → South pole, anticlockwise → North pole

Length Effects:

  • If length doubles with same N: $B$ halves
  • If length doubles with same n: $B$ unchanged
  • Field at ends = half of center field
  • Outside ideal solenoid: $B ≈ 0$
Solenoid Proportionalities:

$B \propto n$ (turns/m), $B \propto I$ (current), $B \propto \mu$ (permeability). If iron core inserted: $B$ increases by $\mu_r$ factor.

MCQ Shortcut:

For solenoid: $B \propto N/L \times I$. If question gives N, L, I changes separately, calculate ratio: $\frac{B_2}{B_1} = \frac{N_2}{N_1} \times \frac{L_1}{L_2} \times \frac{I_2}{I_1}$

Motion of Charged Particles in Magnetic Field

Lorentz Force Equation:

$F = q(v \times B)$ or $F = qvB\sin\theta$ where $\theta$ = angle between v and B.

$\text{Magnetic force: } F = qvB\sin\theta$

Force Direction Rules:

  • Positive charge: Right hand rule (thumb = force, index = field, middle = velocity)
  • Negative charge: Left hand rule OR reverse direction of positive charge
  • Force is perpendicular to both v and B

Types of Motion:

Angle θ Force Motion Path
0° or 180° F = 0 Straight line No deflection
90° F = qvB (max) Circular Circle
Other angles F = qvBsinθ Helical Helix
Important Note:

Magnetic force does NO work on charged particle (F ⟂ v). Therefore, kinetic energy and speed remain constant. Only direction changes.

Charged Particle Motion MCQs (25)

Force on Current Carrying Conductor

Force Equation:

$F = BIL\sin\theta$ where θ = angle between conductor and magnetic field.

$\text{Force on conductor: } F = BIL\sin\theta$

Fleming's Left Hand Rule:

Thumb = Force, Index finger = Magnetic Field (N→S), Middle finger = Current (conventional + to -)

Special Cases:

  • θ = 0° (conductor ∥ field): F = 0
  • θ = 90° (conductor ⟂ field): F = BIL (maximum)
  • θ = 30°: F = BIL × 0.5
  • θ = 60°: F = BIL × 0.866
Force Proportionalities:

$F \propto B$, $F \propto I$, $F \propto L$, $F \propto \sin\theta$. If any two factors double, force quadruples (if others constant).

Magnetic Flux:

$\phi = B \cdot A = BA\cos\theta$ where θ = angle between B and area vector (normal to surface).

  • Maximum flux: θ = 0°, cos0° = 1, $\phi_{max} = BA$
  • Minimum flux: θ = 90°, cos90° = 0, $\phi = 0$
  • Unit: Weber (Wb) = Tm²
Common MCQ Pattern:

Given: F = BILsinθ. If B doubles, I halves, L triples, and θ changes from 30° to 60°, find new F. Solution: $F_{new} = F_{old} \times \frac{B_{new}}{B_{old}} \times \frac{I_{new}}{I_{old}} \times \frac{L_{new}}{L_{old}} \times \frac{\sin\theta_{new}}{\sin\theta_{old}}$

Equations and Their Interrelations

All Important Equations:

$1.\ B_{\text{wire}} = \frac{\mu_0 I}{2\pi r}$
$2.\ B_{\text{solenoid}} = \mu_0 n I = \frac{\mu_0 N I}{L}$
$3.\ F_{\text{charge}} = qvB\sin\theta$
$4.\ F_{\text{conductor}} = BIL\sin\theta$
$5.\ \text{Radius: } r = \frac{mv}{qB}$
$6.\ \text{Period: } T = \frac{2\pi m}{qB}$
$7.\ \text{Magnetic flux: } \phi = BA\cos\theta$
$8.\ \text{Force between wires: } \frac{F}{L} = \frac{\mu_0 I_1 I_2}{2\pi d}$

Equation Manipulation Trick:

When multiple factors change, write ratio: $\frac{B_2}{B_1} = \frac{I_2}{I_1} \times \frac{r_1}{r_2}$. Cancel common factors. Same for other equations.

Equations MCQs (20)

Circular Motion in Uniform Magnetic Field

Circular Motion Equations:

Magnetic force provides centripetal force: $qvB = \frac{mv^2}{r}$

$\text{Radius: } r = \frac{mv}{qB} \quad \text{Period: } T = \frac{2\pi m}{qB} \quad \text{Frequency: } f = \frac{qB}{2\pi m}$

Important Proportionalities:

  • $r \propto m$ (heavier = larger circle)
  • $r \propto v$ (faster = larger circle)
  • $r \propto 1/q$ (more charge = smaller circle)
  • $r \propto 1/B$ (stronger field = smaller circle)
  • $T \propto m$ (heavier = longer period)
  • $T \propto 1/q$ (more charge = shorter period)
  • $T \propto 1/B$ (stronger field = shorter period)
  • Important: $T$ independent of $v$!

Comparison of Particles:

Particle Mass (m) Charge (q) Radius (r) Period (T) Force (F)
Proton m +e r ∝ m T ∝ m F = evB
Electron m/1840 -e r ∝ m/1840 T ∝ m/1840 F = evB (opposite dir)
Alpha 4m +2e r ∝ 4m/2e = 2m/e T ∝ 4m/2e = 2m/e F = 2evB
MCQ Solving Method:

For comparison questions: Write ratio. Example: $\frac{r_\alpha}{r_p} = \frac{m_\alpha/q_\alpha}{m_p/q_p} = \frac{4m/2e}{m/e} = 2$. So alpha radius = 2 × proton radius.

Helical Motion:

When velocity at angle θ ≠ 90° to B field:

  • $v_\perp = v\sin\theta$ causes circular motion
  • $v_\parallel = v\cos\theta$ causes linear motion
  • Radius: $r = \frac{mv_\perp}{qB} = \frac{mv\sin\theta}{qB}$
  • Pitch: $p = v_\parallel T = \frac{2\pi mv\cos\theta}{qB}$
